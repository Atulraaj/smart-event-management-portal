# Smart Event Management Portal

Tech Stack:

* Java 21
* Spring Boot 3
* PostgreSQL
* Thymeleaf
* Bootstrap 5
* Spring Security
* Maven

Base Package:
com.atul.event.management

Project Structure:

* entity
* repository
* service
* controller
* security
* config
* util

Rules:

* Never create a new project.
* Never change the package name.
* Never rename existing files.
* Always use Java 21.
* Always use PostgreSQL.
* Generate code compatible with Spring Boot 3.
* Put entities in entity package.
* Put repositories in repository package.
* Put controllers in controller package.
* Put services in service package.
* Use Lombok where appropriate.
* Use Thymeleaf for frontend pages.
* Ask before modifying pom.xml.
